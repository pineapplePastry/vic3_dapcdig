# Overview
**Diplomatic Actions Properly Convert Devout Interest Groups (DAPCDIG)** is a mod for Victoria 3.
## Steam Page
https://steamcommunity.com/sharedfiles/filedetails/?id=3639710141
## Description
Using the [Devout Interest Group Conversion Framework](https://steamcommunity.com/sharedfiles/filedetails/?id=3635690757) ([Github](https://github.com/pineapplePastry/vic3_digcf)), this mod replaces two diplomatic actions and adds a third.
## Content
**Change Language of Administration**: Replaced
* Still replaces the primary cultures of a colonial subject with the overlord's own
* Instead of converting 5% of bureaucrats in the target's capital, now converts 50% of bureaucrats across the entire country so that assimilation can begin
* If the subject does not have any variant of Appointed Bureaucrats enacted, it will be
* The assimilation bonus is removed, while the bureaucracy and institution change speed malus is now applied nationwide for the target

**Change Religion of Administration**: New
* Requires target country to be your colonial subject, and have a different state religion
 * If the overlord is not Atheist:
	 * The target's Devout Interest Group will be renamed, and adopt ideologies and traits that are default to that religion (ex. IGs converted to Shinto will have the Pious ideology replaced with Shinto Piousness)
	 * Non-ruler and non-heir characters belonging to the Devout will retire, being replaced by characters of the new religion
	 * If the target is Atheist, they will immediately enact Freedom of Conscience
* If the overlord is Atheist:
	* The target will immediately enact State Atheism
* In both cases:
	* 5% of pops not belonging to the new religion will convert
	* Rulers and Heirs will convert to the new religion
	* 10% of pops belonging to the previous state religion will become radicalised

**Force State Religion**: Replaced
* Same effects as Change Religion of Administration, but activation requirements are unchanged
